=== How to use this template ===
Type the following, replacing ''XXXX'' with the company's stock symbol:

:<code><nowiki>{{asx|XXXX}}</nowiki></code>

;Example
:<code><nowiki>{{asx|CBA}}</nowiki></code>

===Where to use this template===
The best location for this template is in an [[WP:INFOBOX|infobox]] about the company, such as {{tl|Infobox Company}}, at the top of the article, if one exists.  Include this template in parentheses on the same line as the company type:
:<code><nowiki>{{Infobox Company</nowiki></code>
:<code><nowiki>| company_type=[[Public company|Public]] ({{asx|CBA}})</nowiki></code>
:<code><nowiki>}}</nowiki></code>

When no infobox exists, some editors include it in the first sentence of an article about the company:
:<code><nowiki>[[Commonwealth Bank]] is a publically traded company ({{asx|CBA}}).</nowiki></code>

Because redundant links are discouraged, this template should normally only appear in one place in an article about a publicly traded company.
