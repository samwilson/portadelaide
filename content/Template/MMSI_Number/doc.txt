== Purpose ==
Use the ship's [[Maritime Mobile Service Identity|MMSI number]] to create a link to [http://www.marinetraffic.com Marinetraffic.com] where the ship's movements can be tracked. This template also adds the article to [[:Category:MMSI Number]].

This template is intended to be used in the {{para|Ship identification}} field in {{tl|Infobox ship career}}.

== Usage ==

The only parameter in the template is the ship's MMSI number:

<tt><nowiki>
{{MMSI Number|mmsi}}
</nowiki></tt>

Example: for the article on [[Black Watch (ship)|''Black Watch'']], <tt><nowiki>{{MMSI Number|311166000}}</nowiki></tt> results in {{MMSI Number|311166000}}.
